## Como rodar o projeto

Siga os passos abaixo para executar a aplicação em sua máquina:

1. Clone o projeto:
   ```bash
   git clone https://github.com/emilyamorim/Projeto-Minearts.git

2. Inicalizar o ambiente virtual:
   ```bash
   python -m venv venv

3. Ative o ambiente virtual:
   ```bash
   .\venv\Scripts\activate

4. Instale os requerimentos:
   ```bash
   pip install -r requirements.txt

5. Instale as dependências do projeto:
   ```bash
   pip install django

6. Rode o projeto no servidor:
   ```bash
   python manage.py runserver
